# Contributing to CodeSync

Thank you for your interest in contributing to **CodeSync**! 🚀

CodeSync is a professional real-time collaborative IDE that enables developers to code together, collaborate instantly, and build software efficiently. We welcome contributions that improve reliability, collaboration features, performance, security, and developer experience.

---

## ⭐ Star the Repository

If you find this project useful, please consider starring the repository to support its growth.

---

## Ways to Contribute

You can contribute by:

* Fixing bugs
* Improving collaboration features
* Enhancing real-time synchronization
* Optimizing performance
* Strengthening security
* Improving UI/UX
* Improving documentation
* Adding tests and improving code quality

---

## Getting Started

1. Fork the repository.
2. Clone your fork locally.
3. Create a new feature branch.
4. Make your changes and test them.
5. Submit a Pull Request.

---

## Development Guidelines

Please ensure that:

* Changes remain focused on a single issue.
* Code follows the project's existing style and architecture.
* Documentation is updated when necessary.
* New functionality includes appropriate tests where applicable.
* Existing functionality continues to work as expected.

---

## Pull Request Guidelines

Before opening a Pull Request:

* Test your changes locally.
* Perform a self-review.
* Reference the related issue.
* Keep your PR focused and easy to review.

Example:

```text
Fixes #123
```

---

## Testing

Before submitting your changes:

* Ensure existing tests pass successfully.
* Add tests for new features whenever applicable.
* Verify bug fixes with reproducible test cases.
* Consider edge cases and collaborative workflows.

---

## Reporting Issues

When reporting bugs:

* Provide clear reproduction steps.
* Describe expected and actual behavior.
* Include relevant logs or screenshots when applicable.

---

## Community Standards

By participating in this project, you agree to follow the project's Code of Conduct.

Please review `CODE_OF_CONDUCT.md` before contributing.

---

Thank you for contributing to CodeSync and helping build a better collaborative coding experience!
